title: "HideMeFactory"
description: "Deploys HideMeToken instances and maintains an on-chain registry of all confidential tokens. Anyone can create a token."
---

`HideMeFactory` is the entry point for creating confidential tokens. It deploys `HideMeToken` contracts, stores metadata on-chain, and provides paginated querying for the frontend token registry.

**Mainnet address:** [`0x46E16F6E248dfa735D50345b1d2657C8dBC5d60B`](https://etherscan.io/address/0x46E16F6E248dfa735D50345b1d2657C8dBC5d60B)

## createToken

```solidity
function createToken(CreateParams calldata p) external returns (address tokenAddr)
```

Deploy a new `HideMeToken` and register it in the factory. The caller becomes the token owner and receives the initial supply.

### CreateParams

<ParamField path="name" type="string" required>
  Token name (e.g. `"My Private Token"`).
</ParamField>
<ParamField path="symbol" type="string" required>
  Token symbol (e.g. `"MPT"`).
</ParamField>
<ParamField path="initialSupply" type="uint64" required>
  Initial supply minted to the caller in raw units (6 decimals). Pass `0` to start with no supply.
</ParamField>
<ParamField path="observers" type="address[]" required>
  Compliance addresses that can decrypt any holder's balance. Can be an empty array.
</ParamField>
<ParamField path="mintable" type="bool" required>
  If `true`, the owner can call `mint()` to increase supply later.
</ParamField>
<ParamField path="burnable" type="bool" required>
  If `true`, any holder can call `burn()` to destroy their tokens.
</ParamField>
<ParamField path="maxSupply" type="uint64" required>
  Maximum total supply. Pass `0` for unlimited.
</ParamField>
<ParamField path="description" type="string" required>
  Human-readable description stored as metadata.
</ParamField>
<ParamField path="logoUri" type="string" required>
  URI pointing to the token logo image.
</ParamField>
<ParamField path="website" type="string" required>
  Token project website URL.
</ParamField>

Returns the address of the newly deployed `HideMeToken`.

## Registry queries

```solidity
function totalTokens() external view returns (uint256)
function getAllTokens() external view returns (address[] memory)
function getTokensByCreator(address creator) external view returns (address[] memory)
function getTokensPaginated(uint256 offset, uint256 limit) external view returns (TokenInfo[] memory)
```

Use `getTokensPaginated` for the frontend registry to avoid loading all tokens in one call.
